# SpecKit Tools Overview

The SpecKit helpers (`analyze_spec_context`, `load_instructions`, `mark_task_complete`, `parse_tasks`, `validate_build`, `verify_ignore_files`) ship with the mtm-workflow MCP server. They automate feature-spec discovery and checklist management.

All code lives under `mtm-workflow/src/tools/speckit/`. After editing these tools, follow the standard build + robocopy sync process documented in `SETUP-GUIDE.md`.
