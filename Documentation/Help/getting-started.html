Application Overview

The MTM Inventory Application is a comprehensive Work-In-Progress (WIP) inventory management system built with Windows Forms (.NET 8) and MySQL database backend. It provides:

ğŸ¯ Core Features

  • Inventory Management - Add, track, and manage inventory items with parts, operations, and locations
  • Remove Operations - Single and batch removal of inventory items with full audit trail
  • Transfer Operations - Move items between locations with validation and tracking
  • QuickButtons - One-click access to your most recent operations
  • Advanced Features - Bulk operations, Excel import/export, and advanced filtering
  • Transaction History - Complete audit trail with modern search and filtering

First Time Setup

1. System Requirements

Before starting, ensure your system meets the requirements:

  • Operating System: Windows 10/11 (64-bit recommended)
  • Framework: .NET 8 Runtime (installed automatically)
  • Database: MySQL 5.7.24+ or MAMP environment
  • Memory: 4GB RAM minimum, 8GB recommended
  • Storage: 100MB for application, additional space for database

View Detailed Requirements

2. Database Configuration

The application automatically detects your environment and configures the database connection:

Environment Detection: The application uses Model_Users.Database and Model_Users.WipServerAddress properties to automatically determine whether to use Release Mode (mtm_wip_application) or Debug Mode (mtm_wip_application_winforms_test) database.

3. First Launch

When you first launch the application:

  1. Splash Screen - The application will show a loading screen while initializing
  2. Database Connection - Automatic connection to the appropriate database based on environment
  3. User Authentication - Select or create your user profile
  4. Theme Selection - Choose from available themes (Arctic, Default, Fire Storm, Forest, Lavender, Midnight, Mystic, Ocean, Sunset, Urban Bloom)
  5. Main Interface - The tabbed interface will appear with your available operations

Your First Operations

1. Adding Inventory (Inventory Tab)

  1. Click the Inventory tab
  2. Select a Part ID from the dropdown (or type to search)
  3. Choose an Operation from the list
  4. Select a Location where the inventory will be stored
  5. Enter the Quantity (use arrow keys for quick adjustments)
  6. Add any Notes or descriptions if needed
  7. Click Save or press Ctrl + S
Success! Your first inventory item has been added. You'll see confirmation in the status bar and the operation will appear in QuickButtons for easy repeat.

Complete Inventory Guide

2. Viewing Recent Operations (QuickButtons)

  1. Click the QuickButtons tab
  2. See your last 10 operations displayed as clickable buttons
  3. Click any button to repeat that exact operation
  4. Modify quantities or details as needed
  5. Save to complete the repeat operation

QuickButtons Guide

3. Removing Inventory (Remove Tab)

  1. Click the Remove tab
  2. Use Show All to see available inventory
  3. Select items by clicking on rows
  4. Use Ctrl + click for multiple selections
  5. Click Remove Selected and confirm
  6. View transaction history for audit trail
